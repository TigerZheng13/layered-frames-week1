How it works

A page is design + content

Only one of these ever needed an API — and it wasn't the design. Separating them is what makes the no-access workflow possible.

🎨 Design — the AI

  • Layout, colour, cards, typography
  • The weekly timeline structure
  • Highlighted assessments, contact block
  • Needs zero knowledge of your specific course

📋 Content — you

  • Course name, weekly topics, dates
  • Assessment names, due dates, weights
  • Your contact details and office hours
  • Pasted in as plain text — no API needed

Safer than the API, not just easier. The AI is told never to guess academic facts. If you don't supply a due date or a weight, it leaves a visible [ADD LATER] placeholder instead of inventing one — so a made-up date can't slip onto a student-facing page.

Step 2 · Build the page — Method A
✦ Using Claude + the Skill

The guided way

Best if you use Claude. A small "skill" file teaches Claude this exact workflow, so it walks you through it and produces a Canvas-safe page automatically.

1

Import the skill once

Download canvas-page-designer.skill and add it to Claude. You only do this once.

2

Ask for a page

Say: "Design a Canvas homepage for my course." Claude recognises the task and starts the workflow.

3

Pick a style

Claude offers the six styles above. Reply with the one you want — e.g. "the editorial magazine style" or just "#3".

4

Give it your course facts

Claude asks for your weeks, assessments, dates, and contact details. Anything you don't have becomes a clearly-marked placeholder — it never invents.

5

Get paste-ready HTML

Claude returns one block of Canvas-safe HTML plus a short list of any blanks to fill. Preview it in a browser if you like.

Step 2 · Build the page — Method B
✦ Using ChatGPT (or any AI) + the Prompt

The copy-paste way

Best if you use ChatGPT or any other assistant. One ready-made prompt does the same job — no special setup, works anywhere.

1

Open the prompt file

It's a single plain-text file, Canvas-Homepage-Prompt.txt, with instructions and a fill-in-the-blanks form.

2

Fill in your course info

Type your course title, weekly topics, assessments, and contact details into the blanks. Leave anything you don't know — the AI will mark it for later.

3

Pick a style number

The prompt has the same six-style menu (1–6). Keep the one you want, delete the rest.

4

Paste it into your AI

Copy the whole filled-in prompt into ChatGPT (or any assistant) and send. It replies with the HTML.

5

Get paste-ready HTML

Same result as Method A — one block of Canvas-safe HTML, ready to paste.

Step 3 · Put it on Canvas

Paste it in — the same for both methods

No technical skill needed. About two minutes.

1

Open a page to edit

In your course: Pages → + Page (or open an existing page and click Edit).

2

Switch to HTML view

In the editor toolbar, click the </> button. On newer Canvas it may be "Switch to raw HTML Editor" under the ⋮ menu.

3

Paste the HTML

Select everything in that box, delete it, and paste in the HTML the AI gave you.

4

Save

Click Save. The page renders exactly as previewed.

5

Make it the home page (optional)

On the saved page: ⋮ → Use as Front Page, then Course Settings → Choose Home Page → Pages Front Page.

Adding a banner image? Upload it to your course Files first, then insert it with the editor's image button (or paste an <img> tag). Canvas file links work for all your students. The pages look good without one too.
