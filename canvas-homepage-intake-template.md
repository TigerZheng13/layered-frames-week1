# Canvas Homepage — AI Design Intake Template

Fill in the blanks below, then paste the WHOLE thing to any AI assistant.
The AI uses this as its only source of truth — no Canvas access needed.
This replaces the API: you are the data source.

---

## PROMPT TO PASTE INTO THE AI

> Design a Canvas course homepage as a single block of **inline-styled HTML**.
> Rules: use only inline `style="..."` attributes — **no `<script>`, no `<style>` blocks,
> no external fonts/stylesheets** (Canvas strips them). Make it visually polished.
> Do NOT invent any facts — use only the information I provide below. If a field is blank,
> leave a clearly marked placeholder like `[ADD LATER]` rather than guessing.
>
> Style direction I want: **[e.g. cinematic dark theme / clean light / RMIT blue + yellow]**

## AFTER THE AI RESPONDS

1. Copy the HTML it produced.
2. In Canvas: course → **Pages** → new/edit → click **`</>`** (HTML view) → paste → **Save**.
3. For a homepage: page **⋮ → Use as Front Page**, then Settings → Choose Home Page → Pages.
4. Insert your banner image via the editor's image button (upload to Files first).

See `HOW-TO-paste-into-canvas.md` for the full manual-upload guide.

## 中文说明

API 之前只帮我提供了"内容"(Assignment、日期等),并不参与"设计"。
没有 API 时,**由人把这些信息填进上面的模板**,粘给任何 AI 即可 —— 人代替了 API 当数据源。
